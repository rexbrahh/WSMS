# Research (non-runtime)

This tree is for **analysis and experiments only**. The WSMS runtime is Go.

Allowed later:

- Statistical evaluation of forced-reset benchmarks (`scipy` / `pandas` / plots)
- Hugging Face / torch soft-prefix experiments
- Notebooks that **read** Go-exported SQLite/JSONL or frozen capsules

Not allowed:

- Authoritative ledger writes
- Authoritative WSL production
- Capsule rendering that diverges from `internal/renderer`
